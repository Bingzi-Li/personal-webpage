URECA

Undergraduate Research Experience on Campus: VR Assisted Rehabilitation of Adhesive Aapsulitis of Shoulder

keyword: Unity, C#, VR, Inverse Kinematics, Dynamic Mesh Coloring
The aim of my part in this project is to provide the players with real-time visualization of humanoid model using Inverse Kinematics, Mirroring and coloring to represent the stretching and contracting of real muscles, using C# scripting in Unity.

  • Date: September 2018 - June 2019
  • Project Members: individual
  • Category: Research Project

HangOut-SG

Andriod App for Hanging Out with Friends

keyword: Spring framework, Andriod Studio, MySQL database
This is a software engineering school project to create an mobile app for residents in Singapore to go out with friends on weekends. It includes features of Map, shop booking, vendor, customer and admin access control etc. This project got A or above.
I contributed in alomost all parts, including backend API, frontend UI and business logic.

  • Project Members: 6 students
